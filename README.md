# CheckVIES Client Examples

This repository contains examples of using client libraries to access the **CheckVIES** service API.

CheckVIES is a service for validating VAT numbers via the VIES (VAT Information Exchange System) system of the European Commission.

## Repository Structure

Examples are organized by technology and programming language:

* [csharp-netcore](./csharp-netcore) — Example using .NET 10 (C#) with a Background Service (Worker Service).
* [typescript](./typescript-nodejs) — Example using Node.js (TypeScript) with a simple console script.

## How to Use

1. Select your preferred technology from the list above.
2. Navigate to the corresponding folder.
3. Follow the instructions in the `README.md` inside that folder to configure your API key and run the example.
